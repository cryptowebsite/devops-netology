Minikube role
=========

Роль для установки minikube на хостах с ОС: Debian, Ubuntu.

Requirements
------------

Поддерживаются только ОС семейств debian.


Example Playbook
----------------

    - hosts: all
      roles:
         - ./roles/minikube
